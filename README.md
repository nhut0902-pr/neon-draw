# Neon Draw

Ứng dụng vẽ bằng cử chỉ tay (Gesture Drawing) sử dụng MediaPipe Hands.

## Tính năng
- Vẽ bằng ngón tay trên không trung thông qua camera.
- Hỗ trợ nhiều màu sắc neon.
- Các cử chỉ điều khiển:
  - **Trỏ + Giữa thu**: Vẽ.
  - **Pinch (Bóp ngón)**: Di chuyển nét vẽ.
  - **Nắm -> Xòe**: Xóa toàn bộ.
  - **Giữ màu**: Chọn màu mới.

## Triển khai
Dự án này được cấu hình để chạy trực tiếp trên Vercel.

## Công nghệ sử dụng
- HTML5 Canvas
- MediaPipe Hands
- JavaScript (Vanilla)
